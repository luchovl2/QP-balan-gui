# GUI de balanceador de carga con QP

El framework QP provee facilidades para construir una GUI que permite correr el firmware en desarrollo, en Windows, sin tener el hardware.
Este proyecto consiste en una GUI para un balanceador de carga para baterías de Li-po.
Si bien el firmware está destinado a correr en microcontrolador de 32 bits, la GUI permite correrlo en Windows, salvo la capa física, obviamente, y así probar la logíca y demás.
